# Kiro Prompts Used – RetroSnakeA

## Prompt 1 – Base Setup
Generate a basic Snake game using HTML Canvas and JavaScript with grid-based movement.

## Prompt 2 – Movement Logic
Implement snake movement logic with arrow key controls and a game loop.

## Prompt 3 – Collision Detection
Add wall and self-collision detection logic to a snake game.

## Prompt 4 – Food and Growth
Implement food spawning and snake growth when food is eaten.

## Prompt 5 – AI Safety Logic
Write JavaScript logic to simulate next snake moves and detect unsafe directions.

## Prompt 6 – Optimization
Optimize the snake game loop for smoother rendering and better performance.
